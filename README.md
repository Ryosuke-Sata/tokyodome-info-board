# Tokyo Dome Info Board (東京ドーム情報掲示板)

東京ドームのイベントスケジュールと、首都圏の電車運行情報をリアルタイムで表示するデスクトップアプリケーションです。
デジタルサイネージのようなモダンなUIで、常に最新の情報を確認できます。

## 📸 特徴

- **モダンなUI**: `customtkinter` を使用した視認性の高いダークモードデザイン。
- **イベント情報**: 東京ドームの「今日」と「明日」のイベントを自動取得して表示。
- **電車運行情報**: 首都圏の主要路線の遅延情報をリアルタイム表示。
- **非同期更新**: データの更新中も時計やUIが止まることなく動作します。
- **堅牢性**: データ保存時の排他制御により、読み込みエラーやファイルの破損を防ぎます。

## 🛠️ 必要要件

- Windows 10/11
- Python 3.10 以上
- Google Chrome (Seleniumでのスクレイピングに使用)

## 📦 インストールと実行

### 1. 準備
このリポジトリ（またはフォルダ）に含まれるすべてのファイルをダウンロードしてください。

### 2. ライブラリのインストール
`requirements.txt` を使用して、必要なライブラリを一括インストールします。
コマンドプロンプト（またはターミナル）を開き、フォルダへ移動してから以下を実行してください。

```bash
pip install -r requirements.txt
3. プログラムの実行
メインファイルである gui.py を実行します。

Bash
python gui.py
初回起動時、data/ ディレクトリと必要なJSONファイルが自動生成されます。

⚙️ 機能詳細
イベント情報 (Tokyo Dome Event)
東京ドーム公式サイトからスケジュールを取得します。

1時間ごとに自動更新されます。

「公式サイト」ボタンをクリックするとブラウザでページを開きます。

運行情報 (Train Status)
Yahoo!路線情報（関東エリア）から遅延・運休情報を取得します。

10分ごとに自動更新されます。

「↻ 更新」ボタンで手動更新が可能です（バックグラウンド処理）。

平常運転時はその旨を表示し、遅延発生時は赤文字で強調表示します。

🖥️ EXE化の手順 (Windows)
PyInstallerを使用して単一の実行ファイルに変換可能です。

PyInstallerをインストール（未導入の場合）:

Bash
pip install pyinstaller
EXEファイルの作成:

Bash
pyinstaller --noconsole --onefile --collect-all customtkinter gui.py
完了後、dist/gui.exe が生成されます。

⚠️ 注意事項
本アプリケーションはWebスクレイピング技術を使用しています。対象サイト（東京ドーム公式サイト、Yahoo!路線情報）の構造変更により、正しくデータを取得できなくなる可能性があります。

スクレイピングは常識的な頻度（イベント:1時間毎、電車:10分毎）で行っていますが、過度な連打や改造はお控えください。

Google Chromeがインストールされている環境でのみ動作します。
