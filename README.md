# Tokyo Dome Info Board (東京ドーム情報掲示板)

東京ドームのイベントスケジュールと、首都圏の電車運行情報をリアルタイムで表示するデスクトップアプリケーションです。
デジタルサイネージのようなモダンなUIで、常に最新の情報を確認できます。

## 📸 主な機能 (Features)

- **モダンなUI**: `customtkinter` を使用した視認性の高いダークモードデザイン。
- **イベント情報**: 東京ドームの「今日」と「明日」のイベントを自動取得して表示。
- **電車運行情報**: 首都圏の主要路線の遅延情報をリアルタイム表示。
- **非同期更新**: データの更新中も時計やUIが止まることなく動作します。
- **堅牢なデータ保存**: データ保存時に一時ファイルを利用することで、処理が中断されてもファイルの破損を防ぎます。

## 🛠️ 動作環境 (Requirements)

- Windows 10/11
- Python 3.10 以上
- Google Chrome (運行情報の取得にSeleniumを使用するため)

## 📦 導入と実行方法 (Installation & Execution)

### 1. ファイルの準備
このリポジトリ（またはフォルダ）に含まれるすべてのファイルをダウンロードしてください。

### 2. 依存ライブラリのインストール
`requirements.txt` を使用して、必要なライブラリを一括でインストールします。
コマンドプロンプト（またはターミナル）を開き、ファイルが存在するフォルダへ移動してから以下を実行してください。

```sh
pip install -r requirements.txt
```
※ requirements.txt には以下のライブラリが含まれている必要があります:
customtkinter, requests, beautifulsoup4, selenium, packaging, tzdata

### 3. アプリケーションの実行
メインファイルである `gui.py` を実行します。

```sh
python gui.py
```
初回起動時、`data` ディレクトリと必要な `json` ファイルが自動で生成されます。

## 📂 ファイル構成 (File Structure)


## ⚙️ 機能詳細 (Feature Details)

### イベント情報 (Tokyo Dome Event)
- **情報源**: 東京ドーム公式サイト
- **更新頻度**: 1時間ごとに自動更新
- **操作**: 「公式サイト」ボタンをクリックするとブラウザで公式ページを開きます。

### 運行情報 (Train Status)
- **情報源**: Yahoo!路線情報（関東エリア）
- **更新頻度**: 10分ごとに自動更新
- **操作**: 
    - 「↻ 更新」ボタンでいつでも手動更新が可能です（UIをブロックしないバックグラウンド処理）。
    - 「Yahoo!路線」ボタンでブラウザで公式ページを開きます。
- **表示**: 平常運転時はその旨を表示し、遅延発生時は赤文字で強調表示します。

## 🖥️ 実行ファイル(EXE)の作成 (Windows)

`PyInstaller`を使用して、Pythonがインストールされていない環境でも動作する単一の実行ファイルに変換することが可能です。

**1. PyInstallerのインストール (未導入の場合)**
```sh
pip install pyinstaller
```

**2. EXEファイルの作成**
```sh
pyinstaller --noconsole --onefile --collect-all customtkinter --collect-all tzdata gui.py
```
コマンド実行後、`dist`フォルダ内に `gui.exe` が生成されます。

## ⚠️ 注意事項 (Notes)

- 本アプリケーションはWebスクレイピング技術を使用しています。情報取得サイト（東京ドーム公式サイト、Yahoo!路線情報）のウェブサイト構造が変更されると、正しくデータを取得できなくなる可能性があります。
- サーバーに負荷をかけないよう、更新は常識的な頻度（イベント:1時間毎、電車:10分毎）で行っています。手動更新の過度な連打や、更新頻度を上げるような改造はお控えください。
- `train_troubledata.py`が内部でSeleniumを使用しているため、PCにGoogle Chromeがインストールされている必要があります。
